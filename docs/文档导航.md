## 📝 文档维护

### 文档更新规范

- **当前文档** (`docs/` 根目录): 保持最新，定期更新
- **归档文档** (`docs/archive/`): 已完成任务、历史版本
- **开发规范** (`.kiro/steering/`): AI 辅助开发规范，随项目演进更新

### 如何贡献文档

1. 新增功能时同步更新 [DOCUMENTATION.md](./DOCUMENTATION.md)
2. 协议变更时更新协议文档并归档旧版本
3. 完成任务后将相关文档移至 `archive/`
4. 更新 [README.md](./README.md) 和 [docs/README.md](./README.md) 索引
